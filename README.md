# 🚗 Car Price Prediction

A full-stack machine learning project that predicts used car prices using an **XGBoost regression model** served through a **Django web app** with a premium dark UI.

---

## 🌐 Live Demo

Run locally at `http://127.0.0.1:8000/` after following the setup steps below.

---

## ⚙️ ML Pipeline

| Stage | Tool |
|---|---|
| Data Cleaning | pandas, numpy |
| Encoding | `OrdinalEncoder`, `OneHotEncoder` (sklearn) |
| Scaling | `StandardScaler` |
| Model | `XGBRegressor` (XGBoost) |
| Evaluation | MAE, RMSE, R², MAPE |
| Serialization | `pickle` |

**Input Features:**

| Feature | Description |
|---|---|
| `brand` | Car manufacturer |
| `milage` | Odometer reading (miles) |
| `fuel_type` | Gasoline / Diesel / Hybrid / Electric |
| `transmission` | Automatic / Manual / CVT / Dual Clutch |
| `accident` | 0 = No accidents, 1 = At least 1 |
| `tx_speed` | Number of transmission speeds |
| `horse_power` | Engine horsepower (HP) |
| `engine_litres` | Engine displacement (L) |
| `car_age` | Current year − model year |

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/7vaibhav31/Car-Price-Prediction.git
cd Car-Price-Prediction
```

### 2. Create & activate virtual environment

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add the dataset

Place `used_cars.csv` inside `ml/data/`.

### 5. Train the model

```bash
cd ml/src
python train.py
```

This will:
- Clean and preprocess the data
- Train the XGBoost pipeline
- Print evaluation metrics (MAE, RMSE, R², MAPE)
- Save `ml/artifacts/pipeline.pkl`

### 6. Run the web app

```bash
cd ../../web
python manage.py runserver
```

Open **http://127.0.0.1:8000/** in your browser.
